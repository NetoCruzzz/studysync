# StudySync

## Project Overview

StudySync is a social productivity application designed to help students stay accountable for their academic goals. Users can create tasks, track their progress, and share updates with friends or accountability groups.

Unlike traditional task managers, StudySync integrates social features such as posts, comments, and group participation to motivate students and promote collaboration.

Following CRUD (Create, Read, Update, Delete) architecture and will be developed using Agile methodology with iterative development sprints.

## Objectives

The main objectives of this project are:

- Practice the Software Development Life Cycle (SDLC)
- Apply Agile methodologies in a team environment
- Develop a social-based application with CRUD functionality
- Emphasize modular code, testing, and documentation
